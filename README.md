# Event Manager (TS)

Small TypeScript project to manage events and participants.

## What it does

* create / update / delete events
* add / remove participants
* group events by theme (Rock, Classical)


## Install

```bash
npm install
```

## Run

```bash
npx ts-node src/index.ts
```

## Tests (optional)

```bash
npm install --save-dev jest ts-jest @types/jest
npm test
```

## Notes

Runs in console only. No UI.
